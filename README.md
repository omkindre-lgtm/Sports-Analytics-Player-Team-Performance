# Sports-Analytics-Player-Team-Performance

# PROJECT OBJECTIVE
Analyze team & player performance using match data
Track total matches, wins, and points
Identify top-performing players and teams
Provide data-driven insights for sports strategies

# INTRODUCTION

Sports analytics uses data analysis techniques to evaluate player and team performance, helping teams make informed strategic decisions.
This project focuses on analyzing match data to understand team consistency, player contributions, scoring patterns, and overall performance trends.
By using data visualization and key performance metrics, the analysis provides valuable insights that can support coaches, managers, and analysts in improving gameplay strategies, player selection, and team development.


# KEY INSIGHTS
Emma is the standout player across metrics
Warriors are the most consistent winning team
Rangers & Titans are competitive rivals
Lower teams (Falcons & Sharks) need performance improvement

# CONCLUSION
Power BI helped uncover patterns in performance
Insights can help:
Coaches in strategy planning
Managers in player selection
Teams in benchmarking & improvement
